  • Virga — https://github.com/IndicoDataSolutions/virga

  • just-jobs — https://github.com/thearchitector/just-jobs
    • A lightweight asynchronous Python job executor. Using Redis by default (but not exclusivly, via custom adapters), it is a smaller and production-ready alternative to Celery for applications where distributed microservices are overkill.
  • MAIS™ — https://github.com/sdbase/mais-access
    • MAIS™ Accounts and MAIS™ Access form a secure, extensible, and suite-wide user authentication and management system (ACL) built on SSL and JWT.
  • webpacker-pnpm — https://github.com/thearchitector/webpacker-pnpm
    • A Ruby gem replacing Rails' default Yarn environment with pnpm, encouraging better code through greater dependency awareness and management.
  • Practical Pig — https://github.com/thearchitector/practical-pig
    • Practical Pig is an opinionated Ruby on Rails web application template that uses webpacker-pnpm, RuboCop, and Prettier to provide a simple yet robust foundation, based on the guiding principle "only require stuff you'll actually use."
